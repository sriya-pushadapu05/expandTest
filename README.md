# ExpandTest Automation Framework

## Project Description

This project is a Selenium Java Automation Framework developed using Page Object Model (POM) for a Notes web application. It automates user registration, login, and note management functionalities like creating, editing, deleting, and filtering notes.

## Features

* User Registration and Login Automation
* Create, Edit, Delete Notes
* Filter Notes by Category
* Search Notes
* Form Validations
* Screenshot on Failure
* Extent Reports

## Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* ExtentReports

## Reports

Execution reports are generated in the `/reports` folder using ExtentReports.

## Screenshots

Failure screenshots are captured automatically and stored in the `/screenshots` folder
